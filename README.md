# Madame Zorblax: Mystik Tarot Zone

## Overview

Madame Zorblax: Mystik Tarot Zone is a deliberately chaotic and poorly designed web experience created for the *Ugly Web Challenge 2026*.

The project simulates a sarcastic, hostile tarot reader who predicts the user’s future while actively mocking them. The website intentionally violates standard UI/UX principles to create confusion, frustration, and visual discomfort, while still remaining technically functional.

## Concept

The core idea is to parody “future prediction” systems through a fictional tarot reader named Madame Zorblax. Instead of providing meaningful insights, the system generates randomized tarot readings combined with sarcastic commentary and insults.

The experience blends:

* Dark humor and satire
* Glitch-inspired aesthetics
* Early-internet chaotic design patterns
* Intentional inefficiency and misdirection

## Features

### Tarot Reading System

* Users select from a set of tarot cards
* A random card is revealed with a themed prediction
* Each reading includes:

  * Card name and symbolism
  * A narrative interpretation
  * A sarcastic or insulting remark
  * A final “verdict”

### Intentional UX Failures

* Misleading navigation buttons
* Broken or fake features (e.g., premium access, refund buttons)
* Random interruptions via popups
* Forced waiting through fake loading screens
* Unexpected page reloads

### Visual Design

* Aggressive use of clashing purple and red color schemes
* Mixed and inconsistent typography
* Overlapping and misaligned elements
* Flashing animations and glitch effects
* Distracting motion and excessive visual noise

### Interactive Elements

* Animated tarot card selection and reveal
* Fake “AI prediction meters” with random values
* Crystal ball interactions that provide useless responses
* Randomized insults and commentary
* Dynamic visitor counter simulation

## Technical Details

* Built using HTML, CSS, and JavaScript
* No external frameworks used
* Heavy use of CSS animations and keyframes
* DOM manipulation for dynamic content generation
* Randomization logic for cards, messages, and UI behavior

## How to Use

1. Open the website using the provided live URL
2. Accept the initial popup disclaimer
3. Select a tarot card
4. Wait through the intentionally slow loading process
5. View your “prediction” and endure the commentary
6. Optionally repeat the process for more readings

## Design Philosophy

This project intentionally breaks conventional design rules, including:

* Clarity and readability
* Accessibility standards
* Logical navigation
* Performance optimization
* User comfort

Every frustrating interaction is deliberate and contributes to the overall experience.

## Judging Criteria Alignment

* Horrible user experience:
  Confusing flows, unnecessary steps, and misleading interactions

* Clashing, cringe-worthy colors:
  High-contrast purple, red, and neon palettes

* Pure intentional inefficiency:
  Fake loading systems, redundant actions, and pointless features

## Disclaimer

This project is satirical and designed purely for entertainment within the context of the competition.
All predictions are fictional and intentionally inaccurate.
